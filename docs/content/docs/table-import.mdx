<Callout type="info">
Import CSV, TSV, JSON, and Excel files with automatic column name matching. Choose between append and truncate modes.
</Callout>

## Supported Formats

| Format | Extensions |
|---|---|
| CSV | `.csv` |
| TSV | `.tsv` |
| JSON | `.json` |
| Excel | `.xlsx`, `.xls` |

## Import Workflow

<Steps>
  <Step>
    ### Select Target Table
    Right-click the target table and select "Import Data".
  </Step>
  <Step>
    ### Choose File
    Select the file to import.
  </Step>
  <Step>
    ### Review Column Mapping
    DBX auto-maps file columns to table columns by name. Adjust mappings manually if names don't match. Unmapped columns are skipped.
  </Step>
  <Step>
    ### Select Mode and Import
    Choose the import mode and click "Start Import".
  </Step>
</Steps>

## Import Modes

| Mode | Behavior |
|---|---|
| **Append** | Insert rows alongside existing data |
| **Truncate then import** | Delete all existing rows first, then insert |

<Callout type="warn">
Truncate mode permanently removes all existing data in the target table. Make sure this is the intended behavior.
</Callout>
