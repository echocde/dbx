<Callout type="info">
树形层级浏览数据库中的所有对象，支持搜索、置顶、字段注释显示和连接颜色标记，多连接多环境轻松区分。
</Callout>

## 树形视图

侧边栏以树形层级展示数据库结构：

```
连接
 └── 数据库
      └── Schema
           └── 表 / 视图
                └── 字段（类型、约束、默认值）
```

点击节点展开或折叠，点击表名直接打开数据浏览视图。字段节点会显示数据类型等属性。

## 搜索与置顶

- **搜索** — 在侧边栏顶部输入关键词，可跨连接过滤匹配的表、视图或字段
- **置顶** — 将常用的表或连接置顶显示，不管层级多深都能快速访问

## 右键上下文菜单

右键点击节点，根据类型提供不同操作：

- **表** — 打开数据、查看结构、复制表名、删除表
- **字段** — 复制字段名
- **连接** — 刷新元数据、断开连接

## 字段注释显示

如果数据库中的字段设置了注释（`COMMENT`），树形视图会在字段名旁边显示注释文本，帮助理解字段含义。

## 连接颜色标记

<Callout type="info">
为每个连接设置颜色标记，开发环境用绿色、生产环境用红色，一眼区分，避免误操作。
</Callout>

颜色显示在侧边栏的连接名称旁和标签页中，多连接场景下尤其实用。
