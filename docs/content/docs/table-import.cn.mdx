---
title: 表数据导入
description: 导入 CSV、TSV、JSON、Excel 文件到数据库表，自动列映射。
---

<Callout type="info">
支持 CSV、TSV、JSON、Excel 文件导入，自动匹配列名，可选追加或清空后导入。
</Callout>

## 支持的文件格式

| 格式 | 说明 |
|---|---|
| CSV | 逗号分隔的文本文件 |
| TSV | 制表符分隔的文本文件 |
| JSON | JSON 数组格式 |
| Excel | `.xlsx` 格式 |

## 使用方式

<Steps>
  <Step>
    ### 选择目标表
    在目标表上右键，选择"导入数据"。
  </Step>
  <Step>
    ### 选择文件
    选择要导入的文件。
  </Step>
  <Step>
    ### 确认列映射
    DBX 自动将文件中的列与目标表的字段按名称匹配。你可以手动调整映射关系，不需要导入的列可以取消映射。
  </Step>
  <Step>
    ### 选择模式并导入
    选择导入模式后，点击"开始导入"。
  </Step>
</Steps>

## 导入模式

| 模式 | 说明 |
|---|---|
| **追加** | 将文件数据追加到现有数据之后 |
| **清空后导入** | 先清空表中所有数据，再导入文件内容 |

<Callout type="warn">
清空模式会永久删除目标表中的所有现有数据。请确认这是你的预期操作。
</Callout>
