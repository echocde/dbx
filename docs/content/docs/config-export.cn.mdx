<Callout type="info">
导出你的数据库连接配置（含加密密码），用 AES-256-GCM 加密保护，安全地在不同电脑之间迁移。
</Callout>

## 导出

<Steps>
  <Step>
    ### 发起导出
    点击侧边栏顶部的 **导出** 按钮。
  </Step>
  <Step>
    ### 设置加密密码
    输入密码短语来加密连接密码。
  </Step>
  <Step>
    ### 保存文件
    选择保存位置。导出文件包含所有连接及密码，已加密。
  </Step>
</Steps>

## 导入

<Steps>
  <Step>
    ### 发起导入
    点击侧边栏顶部的 **导入** 按钮。
  </Step>
  <Step>
    ### 选择文件并解密
    选择导出的文件，输入密码短语解密。
  </Step>
  <Step>
    ### 完成
    连接导入完成，密码自动恢复。按 名称 + 主机 + 端口 自动去重。
  </Step>
</Steps>

## 安全性

- 密码使用 **AES-256-GCM** 加密，密钥通过 **PBKDF2** 派生
- 没有密码短语无法解密文件
- 导出文件是标准 JSON 格式，可以通过 U 盘、网盘等方式传输

## 向后兼容

导入同样支持旧版本导出的明文 JSON 文件（不含密码）和原始的连接配置数组。
