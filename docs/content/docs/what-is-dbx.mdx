<Callout type="info">
15 MB install size. 25+ database engines. Built-in AI assistant. Desktop + Docker deployment.
</Callout>

DBX is an open-source, lightweight, cross-platform database management tool built with [Tauri 2](https://tauri.app/) and Rust. No bundled Chromium — just a 15 MB installer that manages MySQL, PostgreSQL, MongoDB, Redis, and 25+ more databases.

![DBX Screenshot](/screenshot-dark.png)

## Why DBX?

<Cards>
  <Card title="Ultra-Lightweight" href="/en/docs/getting-started">
    ~15 MB installer, no Chromium. Tauri 2 + Rust for native performance and instant startup.
  </Card>
  <Card title="25+ Databases" href="/en/docs/databases">
    One tool for MySQL, PostgreSQL, SQLite, Redis, MongoDB, ClickHouse, SQL Server, Oracle, and more.
  </Card>
  <Card title="AI-Powered" href="/en/docs/ai-assistant">
    Built-in AI assistant: natural language to SQL, query optimization, error fixing. MCP Server lets Claude/Cursor query your databases directly.
  </Card>
  <Card title="Secure by Default" href="/en/docs/ssh-tunnel">
    Passwords in system keyring (macOS Keychain / Windows Credential Manager). SSH tunnels and AES-256-GCM encrypted config export.
  </Card>
</Cards>

## Key Features

<Cards>
  <Card title="Query Editor" href="/en/docs/query-editor">
    Powered by CodeMirror 6. Syntax highlighting, smart autocomplete, JOIN suggestions, query history.
  </Card>
  <Card title="Data Grid" href="/en/docs/data-grid">
    Virtual scrolling for 100K+ rows. Inline editing, WHERE/ORDER BY filtering, Copy as INSERT, SQL preview before saving.
  </Card>
  <Card title="Schema Browser" href="/en/docs/schema-browser">
    Tree view with search, pin, context menus, create/drop databases and schemas, column comments, and connection colors.
  </Card>
  <Card title="Schema Diff" href="/en/docs/schema-diff">
    Compare schemas across databases and generate sync SQL. Dev vs Production in one click.
  </Card>
  <Card title="Data Transfer" href="/en/docs/data-transfer">
    Cross-engine migration — MySQL to PostgreSQL with automatic type mapping.
  </Card>
  <Card title="Field Lineage" href="/en/docs/field-lineage">
    Trace column relationships across foreign keys, views, and query history with confidence levels.
  </Card>
</Cards>
