---

## 安装

从 [Releases](https://github.com/t8y2/dbx/releases) 页面下载最新版本。

### Homebrew (macOS)

```bash
brew install --cask t8y2/tap/dbx
```

更新：
```bash
brew upgrade --cask t8y2/tap/dbx
```

### Scoop (Windows)

```bash
scoop bucket add dbx https://github.com/t8y2/scoop-bucket
scoop install dbx
```

更新：
```bash
scoop update dbx
```

### Docker 自托管

你也可以通过 Docker 部署 DBX，适合团队共享和服务器环境：

```bash
docker run -d --name dbx -p 4224:4224 -v dbx-data:/app/data t8y2/dbx
```

部署后通过浏览器访问 `http://localhost:4224` 即可使用。

## 创建第一个连接

1. 点击工具栏的 **新建连接**
2. 选择数据库类型（MySQL、PostgreSQL 等）
3. 填写主机、端口、用户名、密码
4. 点击 **测试** 验证连接
5. 点击 **保存并连接**

## 开发环境搭建

### 环境要求

- [Node.js](https://nodejs.org/) >= 18
- [pnpm](https://pnpm.io/)
- [Rust](https://www.rust-lang.org/tools/install) >= 1.77

#### 系统依赖

**macOS：**

```bash
brew install unixodbc
```

**Linux (Ubuntu/Debian)：**

```bash
sudo apt-get install -y libwebkit2gtk-4.1-dev libgtk-3-dev libappindicator3-dev librsvg2-dev patchelf libssl-dev unixodbc-dev
```

**Windows：**

无需额外安装（ODBC 已内置于 Windows）。

### 运行

```bash
git clone https://github.com/t8y2/dbx.git
cd dbx
pnpm install
pnpm dev:tauri
```

### 构建

```bash
pnpm tauri build
```

安装包输出在 `src-tauri/target/release/bundle/` 目录。
