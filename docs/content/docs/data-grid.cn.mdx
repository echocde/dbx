## 虚拟滚动

数据表格采用虚拟滚动技术，仅渲染当前可视区域内的行，即使表中有数十万条记录也能保持流畅的滚动和操作体验。

## 行内编辑

直接在表格中编辑数据，无需编写 SQL 语句：

- **修改数据** — 双击单元格即可编辑，修改后的单元格会高亮标记
- **新增行** — 点击工具栏中的"新增行"按钮，在表格底部插入一行空记录
- **删除行** — 选中行后点击"删除"按钮，标记该行待删除

所有修改会暂存在本地，不会立即写入数据库。

## SQL 预览

<Callout type="info">
在点击"保存"之前，可以预览 DBX 即将执行的 `UPDATE`、`INSERT`、`DELETE` 语句，确认无误后再提交。
</Callout>

这样可以避免误操作对数据造成不可逆的影响。

## 单元格详情面板

点击单元格可在右侧打开详情面板，提供以下功能：

- **值编辑器** — 在更大的编辑区域中编辑长文本、JSON 等复杂值
- **设为 NULL** — 一键将单元格值设为 `NULL`

## 排序与过滤

- **排序** — 点击列头可按该列升序或降序排列，鼠标悬浮在列头时会显示排序方向提示图标
- **搜索** — 使用搜索框快速查找包含指定文本的行
- **WHERE 过滤** — 输入自定义的 `WHERE` 条件，在数据库层面精确筛选，大表场景比客户端搜索更高效
- **ORDER BY** — 独立的 `ORDER BY` 输入框，在数据库层面控制排序，与 WHERE 过滤互不干扰

## 列宽调整与行号

- 拖拽列头边缘可以调整列宽
- **自适应列宽** — 双击列头边缘，自动根据表头和数据内容调整列宽
- 表格左侧显示行号，方便定位

## 数据导出

| 格式 | 说明 |
|---|---|
| CSV | 逗号分隔的文本文件，适合导入 Excel 或其他工具 |
| JSON | JSON 数组格式，适合程序处理 |
| Markdown | Markdown 表格格式，适合粘贴到文档中 |
| 复制为 INSERT | 右键选中的行，可复制为 `INSERT` 语句，方便粘贴到其他数据库执行 |
