<Callout type="info">
可视化编辑表结构，无需手写 `ALTER TABLE`。所有变更保存前可预览完整的 DDL 语句。
</Callout>

## 列编辑

在表结构编辑器中，通过可视化界面管理表的列：

- **添加列** — 设置列名、数据类型、是否允许 NULL、默认值等属性
- **修改列** — 直接编辑已有列的属性，包括重命名、更改类型、调整默认值
- **删除列** — 选中列后标记待移除

所有修改不会立即生效，而是暂存在本地等待确认。

## 索引管理

在同一界面中管理表的索引：

- 添加新索引，指定索引列和索引类型（普通索引、唯一索引等）
- 修改已有索引的配置
- 删除不再需要的索引

## SQL 预览

<Callout type="info">
在点击"应用"之前，DBX 会展示即将执行的 `ALTER TABLE` 语句，包括 `ADD COLUMN`、`MODIFY COLUMN`、`DROP COLUMN`、`CREATE INDEX` 等。确认无误后再执行。
</Callout>

## 数据库支持

| 数据库 | 支持情况 |
|---|---|
| MySQL | 完整支持 |
| PostgreSQL | 完整支持 |
| SQLite | 完整支持 |
| SQL Server | 完整支持 |

DBX 会根据每种引擎的特性自动适配。例如，SQLite 有限的 `ALTER TABLE` 能力由 DBX 透明处理。
