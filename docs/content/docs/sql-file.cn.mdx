直接执行 `.sql` 文件，智能分割语句（正确处理函数、触发器、字符串中的分号），实时显示进度。
</Callout>

## 执行 SQL 文件

<Steps>
  <Step>
    ### 打开功能
    打开 SQL 文件执行面板。
  </Step>
  <Step>
    ### 选择文件
    选择要执行的 `.sql` 文件。
  </Step>
  <Step>
    ### 选择目标
    选择目标数据库连接。
  </Step>
  <Step>
    ### 执行
    点击"执行"，实时查看进度。
  </Step>
</Steps>

## 智能语句分割

DBX 使用智能分割逻辑来正确解析每条语句的边界：

- **函数和存储过程** — `BEGIN...END` 块不会被错误地按分号拆分
- **触发器定义** — 自定义分隔符的触发器语句能被正确识别
- **引号内的字符串** — 字符串中的分号不会被当作语句分隔符

## 取消执行

SQL 文件较大时，可以随时点击"取消"按钮中断执行。已执行的语句不会被回滚，仅停止后续语句的执行。
